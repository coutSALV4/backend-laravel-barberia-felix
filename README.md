# Backend Barberia Felix

Este proyecto es el backend para la gestión de una barbería, desarrollado con Laravel y MySQL. Proporciona una API segura y escalable para manejar usuarios, roles y autenticación mediante Laravel Sanctum. Incluye funcionalidades como registro y login, administración de barberos y clientes, gestión de citas y control de financiero del negocio.
